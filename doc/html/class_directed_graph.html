Detailed Description

Implements a directed graph. It contains a vertex class and a vertex iterator.

Each vertex in the directed graph contains the adjacency list, and additionaly a list with income connections. Both lists are implemented as vectors, and the vertex contains no information but the vertex ID.

The list of vertex is maintained as an associative array, using C++ stl unordered_map, containing pairs of <vertex_ID, vertex>. A C++ unordered_map uses a hash function over the ID, typically allowing O(1) search.
